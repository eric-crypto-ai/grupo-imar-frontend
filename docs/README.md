## Por qué la docs no vive aquí

El vault Obsidian es indexable, RAG-friendly, sincronizado a iCloud, y centraliza también la documentación de n8n y otros servicios fuera de este repo.

Este repo se mantiene puro: solo código y assets de los frontends.
